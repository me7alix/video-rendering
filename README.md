# Video Rendering

This is my first experience with video rendering.  
I'm using the [YUV4MPEG2](https://wiki.multimedia.cx/index.php/YUV4MPEG2) format because it's easy to generate, and it's supported by both FFmpeg and MPV.

https://github.com/user-attachments/assets/d782cdd0-ed0b-43de-a194-a2e230fadcc4

## Quick start
```
cc -O3 main.c -lm
./a.out
```
